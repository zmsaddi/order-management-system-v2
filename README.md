# نظام إدارة الطلبات المتطور v2.0

نظام إدارة طلبات احترافي متعدد اللغات مبني بـ Vue 3 و TypeScript و Supabase.

## 🚀 الميزات

- **دعم متعدد اللغات**: العربية، الإنجليزية، الفرنسية، الهولندية مع دعم RTL
- **أدوار مخصصة**: لوحات تحكم للمدير، مدير المبيعات، والمندوب
- **طباعة احترافية**: فواتير وتقارير PDF مع دعم العربية
- **أمان متقدم**: حماية على مستوى الصفوف، إدارة الجلسات، تسجيل الأنشطة
- **تصميم متجاوب**: يعمل على سطح المكتب والتابلت والهاتف المحمول
- **تحديثات فورية**: مزامنة البيانات المباشرة عبر التبويبات
- **علامة تجارية مخصصة**: شعار ومعلومات الشركة قابلة للتخصيص

## 🛠️ التقنيات المستخدمة

- **Frontend**: Vue 3 + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Build Tool**: Vite
- **State Management**: Pinia
- **Internationalization**: Vue I18n
- **PDF Generation**: jsPDF + html2canvas
- **Icons**: Heroicons

## 📋 المتطلبات

- Node.js 18+
- npm 8+
- حساب Supabase

## 🔧 التثبيت

1. **استنساخ المستودع**
```bash
git clone <repository-url>
cd order-management-system-v2
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **إعداد البيئة**
```bash
cp .env.example .env
```

حدث ملف `.env` بمعلومات Supabase:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_APP_NAME=Order Management System
VITE_APP_VERSION=2.0.0
```

4. **إعداد قاعدة البيانات**
شغل ملف `database/complete_database_schema.sql` في Supabase SQL Editor

5. **خادم التطوير**
```bash
npm run dev
```

6. **البناء للإنتاج**
```bash
npm run build
```
